# 🚀 将网站部署上线 - 详细步骤

## 方式一：GitHub + Vercel（推荐 ⭐⭐⭐）

### 步骤 1️⃣：上传代码到 GitHub

1. **打开 GitHub 网站**
   - 访问 https://github.com
   - 登录你的账号（如果没有账号，需要先注册）

2. **创建新仓库**
   - 点击右上角的 `+` → `New repository`
   - 仓库名称：`btc-dashboard`（或任何你喜欢的名称）
   - 描述：`比特币价格分析仪表盘`
   - 设置为 **Public**（公开）或 **Private**（私有）都可以
   - ⚠️ **不要**勾选 "Initialize with README"
   - 点击 `Create repository`

3. **推送代码到 GitHub**
   
   在你的项目目录（当前目录）中，打开终端并运行：

   ```bash
   # 添加远程仓库（替换 YOUR_USERNAME 为你的 GitHub 用户名）
   git remote add origin https://github.com/YOUR_USERNAME/btc-dashboard.git
   
   # 推送代码
   git push -u origin main
   ```

   如果遇到身份验证问题：
   - 使用 Personal Access Token（GitHub Settings → Developer settings → Personal access tokens）
   - 或使用 GitHub Desktop 应用

### 步骤 2️⃣：在 Vercel 上部署

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 点击 `Sign Up` 或 `Log In`
   - **选择 "Continue with GitHub"**（用 GitHub 账号登录）

2. **导入项目**
   - 登录后，点击 `Add New...` → `Project`
   - 你会看到你的 GitHub 仓库列表
   - 找到 `btc-dashboard`，点击 `Import`

3. **配置项目**（通常不需要修改）
   - Framework Preset: **Next.js** ✅（自动检测）
   - Root Directory: `./`
   - Build Command: `npm run build`
   - Output Directory: `.next`
   - Node.js Version: 18.x 或更高
   
   **环境变量**（可选）：
   - 暂时不需要配置，使用默认即可

4. **开始部署**
   - 点击 `Deploy` 按钮
   - 等待 1-2 分钟（首次部署可能需要更长时间）
   - 部署完成后，你会得到一个链接，例如：
     ```
     https://btc-dashboard.vercel.app
     ```

5. **🎉 完成！**
   - 点击链接就能访问你的网站
   - 分享链接给任何人，他们都可以访问
   - 每次你推送代码到 GitHub，Vercel 会自动重新部署

## 方式二：使用 Vercel CLI（快速部署）

如果你不想使用 GitHub，可以直接通过命令行部署：

### 步骤 1️⃣：安装 Vercel CLI

```bash
npm install -g vercel
```

### 步骤 2️⃣：登录并部署

```bash
# 登录 Vercel
vercel login

# 部署（第一次会问一些配置问题，全部按回车使用默认值即可）
vercel --prod
```

几分钟后，你会得到一个在线网址！

## 📝 部署后的管理

### 查看访问数据
- Vercel Dashboard → Analytics
- 查看访问量、性能等数据

### 自定义域名（可选）
1. 在 Vercel 项目设置中点击 "Domains"
2. 添加你自己的域名（如 btc.yourname.com）
3. 按照提示配置 DNS 记录

### 更新网站
只需要：
```bash
git add .
git commit -m "更新内容"
git push
```
Vercel 会自动检测并重新部署！

## ❓ 常见问题

**Q: 部署是免费的吗？**
A: 是的！Vercel 的 Hobby 计划完全免费，足够个人项目使用。

**Q: 网站速度会慢吗？**
A: 不会！Vercel 有全球 CDN，访问速度很快。

**Q: 我的网站会一直在线吗？**
A: 是的，只要你的 Vercel 账号正常，网站就会一直在线。

**Q: 如何看到实时访问人数？**
A: Vercel Analytics 可以查看访问数据（需要在项目设置中启用）。

**Q: API 调用有限制吗？**
A: 使用的 CoinGecko 免费 API 有速率限制，但对个人网站足够了。
